Objetivo

O objetivo principal deste tutorial é apresentar como o Material UI for React, biblioteca de componentes React para desenvolver aplicações de modo ágil e fácil, pode ser utilizado para construir seu próprio design.

 

Falando um pouco do Material UI

O que é Material UI?

Em poucas palavras, é um conjunto de diretrizes, também conhecido como guidelines, que foi criado pela Google para padronizar todas as suas interfaces gráficas. A primeira vista, já vemos que uma diretriz do Material Design é a do minimalismo, tudo deve ser muito clean, direto, chamativo, amigável e intuitivo. Além disso, ele tem o propósito de que tudo deve ser como se fosse um mundo real no mundo digital: responsividade imediata a interação do usuário, objetos sólidos, utilização de sombras e luzes, simulação de profundidade, animação baseada em força, onde há aceleraçãi e desaceleração, como acontece na vida real, materiais digitais que não podem passar uns dentro dos outros, etc.

Apresentando o Exemplo

Para mostrar a funcionalidade da biblioteca Material UI, utilizaremos o exemplo do relatório diário. Veja a especificação abaixo:

O sistema deve permitir a apresentação de um relatório diário dos dados de média coletados através das leituras dos sensores presentes no ambiente, contendo as seguintes informações: temperatura (em graus celsius), umidade (em porcentagem), quantidade de água utilizada para regar (em métros cúbicos), eletricidade gasta (em watts).

Preparação do projeto

Para iniciarmos o projeto no Visual Studio Code, os seguintes passos devem ser executados:

  • Descompacte o arquivo material_ui_react_tutorial3.zip em um local de sua preferência. Em seguida, abra o projeto no Visual Studio Code clicando com o botão direito do mouse sobre o diretório do projeto, selecionando "Abrir com outro aplicativo", em seguida selecionando o Visual Studio Code.

  • Agora, abra o terminal do Visual Studio Code, e digite os seguintes comandos: 

    $ npm install
$ npm start

  • Com isso, o browser deve se abrir na página de start default da aplicação. Em seguida, vamos deletar os conteúdos default. Abra o arquivo src/App.js, e remova o código JSX que está disponível na instrução de retorno, assim como a declaração de importação que está importando de ./logo.svg (você também pode excluir este arquivo, pois não é mais necessário).

Veja o código do arquivo App.js 

                                import React, { useState } from 'react';
import { makeStyles } from '@material-ui/core/styles';
import Paper from '@material-ui/core/Paper';
import Grid from '@material-ui/core/Grid';
import Card from '@material-ui/core/Card';
import CardContent from '@material-ui/core/CardContent';
import Typography from '@material-ui/core/Typography';
import sensoresData from './sensores.json';
import img from './background.jpeg';

const useStyles = makeStyles((theme) => ({
  root: {
    flexGrow: 1,
    backgroundImage: `url(${img})`
  },
  paper: {
    padding: theme.spacing(2),
    textAlign: 'center',
    color: theme.palette.text.secondary
  },
  title: {
    padding: 15,
    color: 'white'
  },
  cardList: {
    margin: 5
  }
}));

const semana = {
  1: "segunda",
  2: "terca",
  3: "quarta",
  4: "quinta",
  5: "sexta",
  6: "sabado",
  7: "domingo"
}
export default function FullWidthGrid() {
  const classes = useStyles();
  const [selected, setSelected] = useState(0);
  console.log(sensoresData)
  return (
    <div className={classes.root}>
      <h1 className={classes.title}>Relatório diario</h1>
      <Grid container
        direction="row"
        justify="center"
        alignItems="flex-start"
        spacing={3}>
        <Grid item xs={3}>
          <Paper className={classes.paper}>
            {sensoresData.map((item) => {
              return <Card className={classes.cardList} onClick={() => setSelected(item)}>
                <CardContent >
                  <Typography gutterBottom variant="h5" component="h2">
                    {`Semana: ${item.semana + 1} - ${semana[item.dia]}`}
                  </Typography>
                </CardContent>
              </Card>
            })}
          </Paper>
        </Grid>
        <Grid item xs={3} >
          <Paper className={classes.paper}>
            {selected ? <Card >
              <CardContent>
                <Typography gutterBottom variant="h5" component="h2">
                  {`Semana: ${selected.semana + 1} - ${semana[selected.dia]}`}
                </Typography>
                <Typography variant="h5" component="h2">
                  {`🌡 temperatura: ${selected.sensores.temperatura}`}
                </Typography>
                <Typography variant="h5" component="h2">
                  {`⛲ agua: ${selected.sensores.humidade}`}
                </Typography>
                <Typography variant="h5" component="h2">
                  {`💧 umidade: ${selected.sensores.agua}`}
                </Typography>
                <Typography variant="h5" component="h2">
                  {`⚡ energia: ${selected.sensores.eletricidade}`}
                </Typography>
              </CardContent>
            </Card> : <div>Nao selecionado</div>}
          </Paper>
        </Grid>

      </Grid>
    </div>
  );
}}
